# wtr_diy_joystick
 WTR_DIY_joystick

这是WTR自己制作的手柄，输入为两个模拟摇杆，和一个矩阵键盘。

软件开发基于hal库：

通信包为：包头(2位）+摇杆1的两个通道(8位）+摇杆2的两个通道(8位)+矩阵键盘(1位)
